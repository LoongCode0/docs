# 界面导览

本页介绍 LoongCode 主界面的各个区域，帮你快速熟悉界面布局，找到你需要的功能。

## ① 左侧任务栏

左侧任务栏是管理工作区和任务的主要区域。从上到下依次包含：

- **工作区树**：列出所有已打开的工作区（项目文件夹）。点击工作区名称右侧的**新建任务**图标可在该工作区下新建任务。
- **任务列表**：每个工作区下方展示该工作区的任务条目。每行左侧的彩色圆点表示任务状态（空闲、运行中、成功、出错、提问中），右侧显示未读消息数量角标。
- **搜索**：点击搜索图标可按关键词筛选任务。
- **归档入口**：可查看或管理已归档的历史任务。
- **＋ 新建任务**：栏目底部的**＋ 新建任务**按钮快速在当前工作区新建一个任务。
- **折叠侧栏**：将鼠标悬浮到左上角 Logo 区域，可点击**收起侧栏**将任务栏折叠；点击**展开侧栏**恢复（v0.7.0 起支持）。

另外，左侧栏底部还有**新手引导**快捷入口和**打开设置**按钮。

在这里你可以切换任务、管理工作区，以及快速定位历史对话。详细说明见[任务与工作区](/features/tasks-and-workspaces)。

## ② 顶部标题栏

标题栏横跨界面顶部，包含：

- **当前工作区 / 分支切换器**：显示当前任务所在的工作区名称和 Git 分支（worktree 即 Git 的一种独立检出目录，允许同时在多个分支上工作）；点击分支名称可新建分支、切换分支或新建 worktree。
- **提交菜单**：选择器条（输入框上方）右侧的分支控件打开后，可查看未提交的更改文件数，并进行相关操作。
- **更新徽标**：当检测到新版本时，右上角会出现**更新**徽标；点击后可立即重启完成更新。
- **侧边面板**按钮与**终端**按钮：快捷切换右侧面板和底部终端的显示状态。

在这里你可以快速切换 Git 分支、管理代码版本，以及检查应用更新。

## ③ 中央对话区

中央区域是与 AI 交互的主舞台，显示当前任务的完整对话历史：

- **消息气泡**：用户消息和 AI 回复以时间顺序排列；AI 回复实时流式渲染，Markdown 自动格式化。
- **思考过程块**：AI 的内部推理过程以折叠块形式呈现，标题为「思考过程」；点击可展开或收起。
- **工具卡片**：AI 调用工具（读写文件、执行命令等）时显示带状态的卡片；对代码的修改以 Diff（新旧代码对比）形式展示。
- **提问卡片**：需要用户授权时弹出，包含**允许** / **拒绝**按钮供你选择。
- **分屏**：可同时打开多个对话分栏，每栏独立运行一个任务（各栏有自己的输入框与 AI 会话）。

在这里你可以查看对话历史、审阅 AI 的操作，以及对 AI 的权限请求作出回应。

## ④ 输入框

输入框位于界面底部，是向 AI 发送消息的入口：

- **选择器条**（输入框上方）：从左到右依次为 CLI 选择（Claude CLI / Codex CLI）、模型选择、思考深度（Claude：low / medium / high / xhigh / max / ultracode；Codex：minimal / low / medium / high / xhigh）、执行授权（Claude：default / plan / acceptEdits / bypassPermissions；Codex：只读 / 自动 / 完全访问）、当前分支。
- **上下文用量指示器**：选择器条最右侧显示已用 / 总量的上下文 token 比例；可点击**压缩**对话历史以释放空间。
- **触发器**：在输入框中输入 `@` 可从工作区文件列表添加文件上下文，输入 `/` 可使用内置命令，输入 `$` 可调用已安装的技能。
- **占位提示**：「向 Claude 提问，输入 @ 添加文件，/ 使用命令，$ 使用技能」；等待 AI 回答时变为「请在上方卡片作答；或在此草拟下一条消息…」。

在这里你可以配置每次对话的 AI 参数，以及向 AI 发送指令。详细说明见[对话面板](/features/conversation)。

## ⑤ 右侧面板

右侧面板（点击标题栏**侧边面板**按钮切换显示）包含多个标签页：

| 标签页 | 功能 |
|--------|------|
| 文件树 | 工作区目录结构；可新建、重命名、删除文件与文件夹 |
| 编辑器 | 在应用内直接查看或编辑工作区文件，支持语法高亮与预览 |
| **审查** | 查看工作区的代码 Diff（未暂存 / 已暂存 / 全部分支更改 / 上一轮更改），确认后可提交 |
| **浏览器** | 内嵌浏览器，可在应用内打开网页，并把网页元素直接加入对话上下文 |
| 终端 | 底部可弹出内置终端，直接在工作区目录运行命令 |

::: tip 浏览器标签页
在浏览器标签页中点击 **+** 可新增一个内嵌浏览器窗口；点击**选中元素加入聊天**，可将网页上的元素直接发送给 AI 进行分析。
:::

在这里你可以浏览项目文件、审阅 AI 的代码改动，以及配合内嵌浏览器做前端调试。详细说明见[文件与编辑器](/features/files-and-editor)。

## ⑥ 设置入口

设置入口位于左侧任务栏底部，图标为齿轮形状，点击即可打开设置页面。

设置页面提供以下分类（左侧导航）：

- **常规**：主题、数据目录、CLI 路径、网络代理、提示音、代码快照、自动压缩、使用引导等
- **依赖管理**：检测并安装 Claude CLI / Codex CLI 等依赖
- **模型供应商**：配置各 AI 服务商的 API 密钥（API 密钥是服务商发给你的专属访问凭证）、接口地址、可用模型列表
- **插件**：发现、安装和管理插件
- **技能**：创建或编辑可复用的提示词技能
- **MCP 服务器**：增删改 MCP（Model Context Protocol）服务器配置，扩展 AI 可用的外部工具
- **子智能体**：管理子智能体配置
- **使用统计**：查看 token 用量、费用、活跃热力图等

详细说明见[设置 → 常规](/settings/general)。
