# 浏览器标签页

LoongCode 在右侧面板中内置了一个完整的浏览器，让你无需离开 IDE 就能查阅文档、测试网页，还可以将网页元素直接提供给 AI 使用。

> 相关：[界面导览](/guide/interface-overview)、[对话面板](/features/conversation)

---

## 什么是内嵌浏览器

内嵌浏览器是右侧面板的第三种标签页类型（与文件标签、审查标签并列）。它采用系统原生浏览器内核（Windows 上为 WebView2），支持访问任意真实站点，表现与普通桌面浏览器一致。

**适用场景举例：**

- 查阅 API 文档时，同时在左侧让 AI 写代码
- 把某个网页上的报错信息或 UI 元素截图提供给 AI 分析
- 在不切换窗口的情况下浏览 GitHub Issues 或 StackOverflow 解答

---

## 打开浏览器标签页

1. 点击右侧面板顶部标签栏最右边的 **+** 按钮，弹出下拉菜单。
2. 菜单中选择**浏览器**（另两个选项为「打开文件 Ctrl+P」和「审查」）。
3. 右侧面板切换为浏览器视图，地址栏提示「输入网址后回车」。
4. 在地址栏输入目标网址后按 **Enter** 即可加载页面。

![截图：右侧面板显示浏览器标签页，地址栏已输入 URL，页面正在加载，顶部工具栏显示后退/前进/停止/选元素等按钮](/images/browser_1.png)

::: tip 无协议自动补全
输入时可以省略 `https://`，LoongCode 会自动补充。如果输入内容不像网址（含空格或没有点），会自动以百度搜索兜底。
:::

## 浏览器工具栏

工具栏从左到右依次为：

| 按钮（界面原文） | 说明 |
|---|---|
| **后退** | 返回上一页（当无历史时显示为禁用状态） |
| **前进** | 前进到下一页（当无历史时显示为禁用状态） |
| **停止加载** / 刷新 | 加载中点此停止；加载完成后变为刷新按钮 |
| **地址栏** | 显示当前 URL，可直接编辑后回车导航 |
| **选中元素加入聊天** | 进入「选元素模式」，点击页面元素后将其插入当前对话输入框 |
| **打开调试工具** | 打开独立的原生开发者工具（DevTools）窗口 |
| **关闭浏览器** | 关闭此浏览器标签页 |

## 选取网页元素加入聊天

这是内嵌浏览器的核心功能——你可以直接点选页面上的任何元素（按钮、文字、图片区域等），将其结构化信息插入到当前对话的输入框中，再向 AI 提问。

**操作步骤：**

1. 点击工具栏中的**选中元素加入聊天**按钮，按钮高亮表示已进入选元素模式。
2. 将鼠标移到页面上，目标元素会出现高亮轮廓和信息提示（标签名、颜色、字体等）。
3. 点击想要的元素，该元素的信息（文本内容、HTML 结构、样式、位置等）会以小标签形式（类似 @ 引用文件的样式）插入当前活动任务的输入框光标处。
4. 可以连续点击多个元素，每个元素对应一个独立的小标签。
5. 再次点击**选中元素加入聊天**按钮，或按 **Esc** 键，退出选元素模式。

::: tip 关闭浏览器后标签仍可发送
已选取的元素标签不依赖浏览器标签页保持打开，关闭浏览器后仍可正常发送给 AI。
:::

## AI 回复中的链接就地打开

对话面板中的 Markdown 链接点击后会**在内嵌浏览器中就地打开**，无需离开 LoongCode 切换到系统浏览器。

如果你需要在系统浏览器中打开，可以**按住修饰键**（Ctrl / Cmd）再点击，或使用鼠标中键点击链接。

---

## 多标签与切换

右侧面板支持同时打开多个标签页（文件、审查、浏览器均可并存）。切换到其他标签页后，浏览器页面会在后台**保持加载状态**，切回时页面零重载，登录态和页面内容完整保留。

只有点击浏览器工具栏右侧的**关闭浏览器**按钮（× 图标），才会真正关闭并销毁该浏览器实例。

---

## Agent 使用浏览器的能力

当你在对话中向 AI 提供了网页元素或链接，AI 可以基于这些内容进行分析、代码生成、问题排查等操作。AI 本身不会主动控制浏览器导航——浏览器的操作（地址栏输入、前进后退、选元素等）均由你手动完成后，再将内容提供给 AI。
