多任务并行对话

多个 AI 任务分屏并行，实时流式输出，思考过程、工具调用、代码改动一目了然。

🗂️

工作区与文件管理

按项目组织任务，内置文件树与 Monaco 编辑器，@ 一下就能把文件喂给 AI。

🔀

深度 Git 集成

分支/Worktree 切换、变更审查面板、AI 一键生成提交消息。

🤖

多 CLI 引擎

已支持 Claude Code、OpenAI Codex、OpenCode 三套 CLI，每个任务可独立选择 CLI、模型、思考深度与权限模式，后续持续接入更多。

🧩

可扩展

MCP 服务器、插件市场、技能、子智能体，按需扩展 AI 的能力边界。

📱

跨出桌面

官方 LoongCode App（Android / iOS 原生应用）扫码配对，手机上实时查看与驱动任务；也可接入微信 / 飞书机器人，离开电脑任务照样跑。
