# 会话分享

会话分享（v0.9.8 起，v0.9.10 新增「包含工具」与「密码保护」选项）可以把一段对话中**你挑选的消息**生成一个只读网页链接，发给同事或朋友——对方在浏览器里直接查看排版好的对话内容，无需安装 LoongCode。可以设置有效期与密码保护，也可以随时手动撤销。

> 相关基础：[对话面板](/features/conversation)

## 创建分享

1. 在任意一条 AI 回复底部的操作条中点击**分享**按钮（分叉按钮旁边）。
2. 对话区进入**选择模式**：可分享的消息左侧出现勾选框，**默认全部勾选**。
   - **用户消息**与**每回合的 AI 正文**是各自独立的勾选单元，可以只分享其一；
   - 思考过程与子智能体对话**不参与分享**（显示为置灰），也**不会被上传**；
   - 工具调用卡片默认置灰不参与分享，但可通过底部的「包含工具」选项将其纳入。
3. 在底部浮动操作栏中完成设置：
   - **全选**复选框与「已选 N 条消息」计数；
   - **包含工具**复选框（v0.9.10，默认不勾）：勾选后，被勾选回合的主线工具调用以只读行形态随分享上传——状态符 ✓/✗ + 工具名 + 目标 + 摘要（Edit 显示 `+N −M` 增删行数、错误显示首行），不包含 input/result 全文、思考与子 Agent 内部工具。勾选后选择器中已勾选回合的工具段解除置灰，直观提示哪些将被分享；
   - **密码保护**复选框（v0.9.10，默认不勾）：勾选后系统自动生成 32 位随机密码附在链接中。收到完整链接的人免密直达；只拿到裸链接的人需在分享页手动输入密码才能查看。密码验证通过后记入浏览器本地，同浏览器再次访问无需重输。两个选项**正交**，可同时开启；
   - **有效期**：1 天 / 3 天 / 7 天，默认 1 天；
   - 点击**创建分享链接**（未勾选任何消息时按钮禁用）。
4. 创建成功后弹出**分享已创建**小窗：
   - 链接明文展示，可**复制链接**或直接**打开**；
   - 若开启了密码保护，窗口额外展示密码原文并提供单独的**复制密码**入口——链接与密码可分开告知对方；
   - 小窗同时注明到期时间，以及可在左下角「已分享」中随时撤销。

按 `Esc` 或点击**取消**可退出选择模式；切换任务、关闭分栏也会自动退出。

::: tip 全选联动
全选状态下取消勾选某一条，只会让「全选」框回退为未勾选，**不会**级联取消其他消息；重新勾上「全选」则一键选中全部。
:::

::: tip 「包含工具」的作用范围
工具行跟随所在回合的 AI 正文单元勾选，不单独成条；纯工具回合（本回合 AI 只调用工具而无文本回复）也会产出可勾选的 ghost 单元，不会静默丢失。不分享答案就不分享过程：不勾选「包含工具」时行为与之前版本完全一致。
:::

## 对方看到什么

拿到链接的人在浏览器中打开，看到的是一个**只读**的对话页：

- 标题区显示工作区名与任务标题，及分享时间、消息条数、失效时间等元信息；
- 用户消息以气泡呈现，AI 回复按 Markdown 渲染（含代码高亮）；
- 若创建时开启了「包含工具」，工具调用以只读行形态（状态符 + 工具名 + 目标 + 摘要）展示在同回合的「用户消息之后、AI 正文之前」，同一回合的工具行聚为一组；
- 页面不提供任何交互入口，也不暴露你的其它任务或文件。

链接**过期、被撤销或不存在**时，统一显示「分享不存在或已过期」页面。

若链接带有密码保护，访问时若未携带正确密码，页面提示输入密码；密码通过服务器验证后才会返回内容。

## 管理已分享

左侧栏**底部的「已分享」入口**（分享图标，带有效分享数角标）打开管理弹窗，跨任务列出你创建过的所有分享：

- 每行显示任务标题、「工作区 · N 条消息」与剩余有效时间；
- 行尾操作：**复制**（链接）、**打开**（默认浏览器）、**撤销**；若开启了密码保护，行首有小锁标，复制与打开均使用带密码的完整链接；
- **撤销**需二次确认，撤销后链接**立即失效，无法恢复**；
- 已过期的行置灰，仅保留**清除记录**（只删除本地列表中的这一行）。

## 隐私与安全

::: warning 分享前请确认内容
- 只有你**勾选的用户消息与 AI 正文**会被上传；思考过程与子智能体对话一律不上传。
- **工具调用**（v0.9.10）：仅在勾选「包含工具」后才会随分享上传，且仅上传工具名、目标与摘要等只读描述信息；工具的完整输入输出（input/result）与子 Agent 内部工具一律不上传。
- 图片等附件不会上传，仅以文本占位呈现。
- **链接即凭证**：任何拿到链接的人在有效期内都能查看。建议使用**密码保护**（v0.9.10）增加一道访问门槛——裸链接与密码分开告知，即使链接意外泄露也无法直接查看内容。不再需要时及时撤销。
:::

分享默认走官方分享服务；团队也可以自行部署分享服务器，并在**设置 → 分享**中填写自己的服务器地址与访问密钥，详见[分享设置](/settings/share)。
