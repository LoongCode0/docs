快捷键一览 ​

macOS 用 Cmd 替代 Ctrl；下表以 Windows/Linux 写法为准。

全局快捷键 ​

任意界面下均可触发，无需焦点落在特定区域。

快捷键功能
Ctrl+N新建草稿分栏并选中
Ctrl+Shift+T复活最近关闭的分栏
Ctrl+P打开文件搜索面板
Ctrl+`开关底部终端面板

文件树快捷键 ​

焦点在左侧文件树区域时生效。

快捷键功能
Ctrl+C复制选中的文件/目录
Ctrl+X剪切选中的文件/目录
Ctrl+V粘贴到当前选中项所在目录
Delete删除选中的文件/目录

说明： 右键菜单中的"重命名"条目显示 F2 提示，但 F2 键盘直接触发功能尚在开发中，目前请通过右键菜单执行重命名操作。

编辑器快捷键 ​

代码编辑器（Monaco 编辑器）中生效。

快捷键功能
Ctrl+S保存当前文件

Monaco 编辑器本身还内置了大量通用编辑快捷键（如代码折叠、跳转定义等），可在编辑器右键菜单的"命令面板"中查看完整列表。

文本编辑通用快捷键 ​

由操作系统或底层控件提供，可在输入框等文本区域直接使用。

快捷键功能
Ctrl+C复制选中文本
Ctrl+X剪切选中文本
Ctrl+V粘贴文本
Ctrl+A全选
Ctrl+Z撤销
