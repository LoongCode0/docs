# 深度搜索

深度搜索（v0.9.8 起）在侧栏「按标题筛选任务」的基础上更进一步：**跨全部工作区检索任务标题与会话内容**，命中结果以高亮片段列出，点击即可进入对应任务并**滚动定位到那条消息**。想找「上次让 AI 改过某个函数的那次对话」，再也不用逐个任务翻历史。

> 相关基础：[任务与工作区](/features/tasks-and-workspaces#搜索任务)

## 打开深度搜索

两种方式：

- 点击侧栏搜索框右侧的**深度搜索**按钮（放大镜加文本行的图标，悬浮提示「深度搜索（标题与会话内容）」）。若搜索框里已输入了关键词，会自动带入弹窗作为初始查询。
- 任意界面按 **Ctrl+Shift+F**（macOS 为 **Cmd+Shift+F**）。

弹窗顶部是搜索输入框（自动聚焦）与**包含归档**勾选项，下方为结果列表。按 `Esc` 或点击弹窗外部即可关闭。

---

## 检索范围与匹配规则

| 维度 | 规则 |
|------|------|
| **任务标题** | 全部任务（不分 CLI）都参与标题匹配 |
| **会话内容** | 检索**用户消息、AI 回复与思考过程**的文本；Claude 与 Codex 任务支持内容检索，**OpenCode 任务目前仅参与标题匹配** |
| **不检索** | 工具调用的输入 / 输出、子智能体内部对话 |
| **多个关键词** | 用空格分隔，视为「同时包含」——所有词都出现在**同一条消息**（或同一个标题）里才算命中 |
| **大小写** | 不敏感 |
| **归档任务** | 默认不搜；勾选**包含归档**后参与，命中的归档任务带「已归档」徽标 |

---

## 结果的读法

结果**按任务分组**（类似 VS Code 的全局搜索）：

- **组头**：任务标题（命中部分高亮）+ CLI 徽标 + 「已归档」徽标（仅归档任务）+ 最近活动的相对时间。点击组头 = 仅打开该任务。
- **片段行**：角色小图标（用户 / 助手 / 思考）+ 命中消息的上下文片段，关键词高亮。每个任务最多列出 5 条片段。

结果过多时列表底部会提示「结果已截断，请细化关键词」——加个更具体的词再搜即可。

### 键盘操作

| 按键 | 功能 |
|------|------|
| `↑` / `↓` | 在组头与片段之间上下移动高亮 |
| `Enter` | 打开当前高亮项 |
| `Esc` | 关闭弹窗 |

---

## 跳转定位

点击某条**片段**后，弹窗关闭，LoongCode 会在当前分栏打开该任务，自动滚动到命中的那条消息并**闪烁两下**提示位置；侧栏也会同步展开并定位到该任务所在分组。

::: tip 定位偶尔会「退化」为只进任务
搜索结果基于磁盘上的会话快照。如果点击时该会话恰好被[编辑重跑](/features/conversation#编辑历史消息重跑)、回滚或分叉过，原来的消息位置可能已不存在——此时会安全地退化为只打开任务、不做定位，不会闪烁到错误的消息上。
:::
