# 集成终端

集成终端让你无需切换到外部终端工具，直接在 LoongCode 内运行命令行操作。它是真实的 PTY 终端——PTY 即伪终端，是操作系统提供的终端仿真接口。这意味着 `vim`、`top`、`npm install` 这类交互式程序都能正常运行。

> 相关基础：[界面导览](/guide/interface-overview)

---

## 打开终端

**方法一**：点击顶部标题栏的**终端**按钮。

**方法二**：按快捷键 **Ctrl+`**（反引号，Tab 键上方）。

终端面板从中央对话区底部弹出，覆盖在对话区下方，不影响右侧面板的正常使用。再次点击**终端**按钮或按 Ctrl+` 可关闭面板。终端配色会**跟随当前应用主题**（深色 / 浅色 / 护眼）自动切换（v0.9.1 起）。

::: tip 想在终端里继续 AI 对话？
本页的集成终端是工作区的 **Shell 终端**。若你想把 **AI 对话本身**切换到 CLI 原生的命令行交互界面里继续（续接同一会话），请使用分栏标题栏的[终端 TUI 模式](/features/tui-mode)（v0.9.8 起）。
:::

## 多标签页管理

终端面板支持同时开启多个终端会话：

- 点击面板顶部的**新建终端**按钮（+ 图标），新开一个独立的终端标签页。
- 点击标签页右侧的关闭按钮可**单独关闭**该终端，同时结束其子进程树，不影响其他标签页。
- 点击面板右上角的**关闭终端面板**按钮，关闭并结束当前工作区的**所有**终端。

::: tip
关闭终端会连同该终端启动的所有子进程一起结束，避免后台进程残留占用目录。
:::

## AI 与终端联动

AI（Agent）执行命令时会在当前工作区的终端中运行，你可以实时看到每条命令的完整输出——构建日志、测试结果、安装进度都逐行显示。

你同样可以在终端里直接输入命令，与 AI 协同工作：比如 AI 写好脚本后，你手动在终端里运行并把结果告诉 AI。

## 终端中的可点击链接（v0.8.0 起）

终端输出里出现的网址（如构建日志中的本地服务地址、文档链接）会被识别为可点击链接：

- **左键单击**：直接打开链接。本地地址（如 `localhost` / 内网地址）会在 LoongCode 内置的[浏览器标签页](/features/browser)中打开，便于就地预览；其它网址在系统默认浏览器中打开。按住 `Ctrl`（macOS 为 `Cmd`）单击可强制改用系统浏览器打开。
- **右键单击**：弹出菜单，可选择**复制链接地址**、**在侧边栏浏览器中打开**或**在默认浏览器中打开**；若当前有选中的文本，菜单还会提供**复制**选中内容。

::: tip
右键菜单依赖鼠标悬停先命中链接。通过键盘上下文菜单键（如 `Shift+F10`）在链接上唤起菜单时不会拦截，属正常行为。
:::

## 环境变量

终端继承工作区的环境变量配置。如需添加全局环境变量（对所有任务生效），请前往**设置 → 常规**进行配置；详见[设置 → 常规](/settings/general)。
