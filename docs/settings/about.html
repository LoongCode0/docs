关于 ​

入口： 侧边栏 → 设置 → 关于（位于设置侧栏最底部）

关于页汇总了版本信息、更新检查、用户协议与隐私协议、官方仓库链接、赞助支持与联系方式。

版本信息与检查更新 ​

页面顶部居中显示应用 Logo、名称与当前版本号（v 前缀 + 版本号）。

检查更新行是应用内自动更新的另一个入口（与右上角的更新徽标共用同一套机制，完整流程见安装 → 自动更新），按当前状态显示以下几种样式之一：

状态界面表现
默认（未检查）显示当前版本号 + 检查更新按钮
检查中按钮变为检查中…（旋转图标），不可重复点击
已是最新手动点击检查后若无新版本，显示绿色「✓ 已是最新版本 · 当前 vX」
发现新版本，下载中显示「发现新版本 vX · 正在后台下载 N%」，下方有下载进度条
新版本已就绪显示「新版本 vX 已就绪」+ 立即重启更新按钮
检查失败红色「检查更新失败，请检查网络后重试」+ 重试按钮

TIP

应用启动后会自动在后台检查更新，无需手动操作；本行的检查更新按钮用于随时手动触发一次检查。

用户协议与隐私协议 ​

点击用户协议或隐私协议行进入对应的协议明细页，回退按钮与标题同行显示，右上角标注最后更新日期。协议正文随应用当前语言（中/英）自动切换。点击左上角返回回到关于页列表。

官方仓库 ​

点击官方仓库行，使用系统默认浏览器打开 LoongCode 的 GitHub 发布仓库。

赞助支持 ​

点击赞助支持行展开手风琴面板，并排显示微信与支付宝收款码，可截图或扫码赞助开发者；再次点击收起。

联系方式 ​

点击联系方式行会把支持邮箱地址复制到剪贴板（不会拉起本机邮件客户端），并弹出「邮箱已复制」提示。
