集成终端 ​

集成终端让你无需切换到外部终端工具，直接在 LoongCode 内运行命令行操作。它是真实的 PTY 终端——PTY 即伪终端，是操作系统提供的终端仿真接口。这意味着 vim、top、npm install 这类交互式程序都能正常运行。

相关基础：界面导览

打开终端 ​

方法一：点击顶部标题栏的终端按钮。

方法二：按快捷键 Ctrl+`（反引号，Tab 键上方）。

终端面板从中央对话区底部弹出，覆盖在对话区下方，不影响右侧面板的正常使用。再次点击终端按钮或按 Ctrl+` 可关闭面板。

多标签页管理 ​

终端面板支持同时开启多个终端会话：

  • 点击面板顶部的新建终端按钮（+ 图标），新开一个独立的终端标签页。
  • 点击标签页右侧的关闭按钮可单独关闭该终端，同时结束其子进程树，不影响其他标签页。
  • 点击面板右上角的关闭终端面板按钮，关闭并结束当前工作区的所有终端。

TIP

关闭终端会连同该终端启动的所有子进程一起结束，避免后台进程残留占用目录。

AI 与终端联动 ​

AI（Agent）执行命令时会在当前工作区的终端中运行，你可以实时看到每条命令的完整输出——构建日志、测试结果、安装进度都逐行显示。

你同样可以在终端里直接输入命令，与 AI 协同工作：比如 AI 写好脚本后，你手动在终端里运行并把结果告诉 AI。

环境变量 ​

终端继承工作区的环境变量配置。如需添加全局环境变量（对所有任务生效），请前往设置 → 常规进行配置；详见设置 → 常规。
