前置依赖 ​

为什么需要 CLI ​

LoongCode 是一个图形外壳，它本身不直接调用 AI 模型——真正负责模型交互、代码读写和命令执行的是本机安装的命令行工具（CLI）。

  • Claude Code CLI（claude）：由 Anthropic 提供，是 LoongCode 的必要依赖。所有 Claude 系列模型的对话均通过它完成。
  • Codex CLI（codex）：由 OpenAI 提供，是可选依赖。如果你希望使用 OpenAI 的 Codex 模型，则需要额外安装它。

LoongCode 启动时会自动在系统 PATH（操作系统用于查找可执行程序的目录列表）中搜索这两个命令，找到后即可使用。

如果你是第一次接触这些概念

「CLI」全称 Command Line Interface，是一种通过文字命令操作的程序，通常在终端（黑色窗口）中运行。LoongCode 帮你省去了直接在终端输入命令的麻烦，但这些程序本身仍需要安装在你的电脑上。

方式一：应用内一键安装（推荐） ​

LoongCode 内置了依赖管理功能，无需手动在命令行中操作。

  1. 打开 LoongCode，点击左下角的设置图标，进入设置页面。
  2. 在左侧导航中点击依赖管理。
  3. 页面会自动检测当前系统中已安装的依赖，并列出检测结果：
    • 必须依赖 分组中包含 Claude CLI、Git 等必要工具。
    • 可选依赖 分组中包含 Codex CLI 等可选工具。
  4. 对于显示未安装的依赖项，点击该卡片右侧的安装按钮，LoongCode 将自动完成下载和安装。
  5. 安装完成后，若弹出「依赖已安装」对话框，点击立即重启以使新安装的工具生效。

如果点击「重新检测」后仍显示未安装

这是正常现象。LoongCode 在启动时会固定当前的系统 PATH，新安装的工具需要在重启后才能被检测到。点击立即重启即可解决。

安装来源选择 ​

若你的网络无法直连访问 GitHub，可以在依赖管理页面调整安装来源：

来源选项适用场景
官方脚本默认选项，走 Anthropic / OpenAI 官方安装渠道
GitHub 直连直接从 GitHub 下载，适合能访问 GitHub 的网络环境
GitHub 加速通过镜像加速下载，适合访问 GitHub 较慢的国内网络

在对应依赖卡片的安装来源下拉框中选择合适的选项，然后再点击安装。如需自定义镜像前缀，选择「GitHub 加速」后会出现自定义镜像前缀输入框。

方式二：手动安装 ​

如果你更熟悉命令行，也可以直接从官方渠道手动安装：

Claude Code CLI（必装）：

  • 官方安装文档：https://docs.anthropic.com/zh-CN/docs/claude-code/overview
  • 安装完成后，在终端运行 claude --version 验证是否成功。

Codex CLI（可选）：

  • 官方安装文档：https://github.com/openai/codex
  • 安装完成后，在终端运行 codex --version 验证是否成功。

手动安装好 CLI 工具后，重新启动 LoongCode，它会自动在 PATH 中找到这些命令。

手动指定 CLI 路径 ​

如果你的 CLI 工具安装在非标准路径，或者你希望使用特定版本的命令，可以手动告诉 LoongCode 去哪里找。

  1. 打开设置，点击左侧导航中的常规。
  2. 找到 Claude CLI 命令 或 Codex CLI 命令 设置项。
  3. 在输入框中填写命令名（如 claude）或完整的可执行文件路径（如 /usr/local/bin/claude）。
    • 填写命令名时，LoongCode 会经由系统 PATH 自动解析为绝对路径。
    • 留空则恢复自动发现模式。
    • 也可以点击选择文件按钮，通过文件选择对话框浏览定位可执行文件。

确认路径已生效

保存设置后，输入框下方会出现已解析提示并显示找到的完整路径，说明 LoongCode 已成功找到 CLI 工具。若显示未找到，请检查路径是否正确。

账号与 API 准备 ​

安装好 CLI 工具后，还需要有对应的账号或 API 密钥才能真正发起对话。API 密钥是服务商发给你的一串专属访问凭证，类似密码，用于确认你的身份并计算费用。有两种方式：

方式一：使用 CLI 官方账号登录

在终端中运行对应的登录命令（如 claude login），按提示完成授权，登录后 CLI 会自动保存凭证，LoongCode 调用时会直接使用。

方式二：配置第三方模型供应商

LoongCode 支持接入第三方 AI 服务商（如 DeepSeek 等），无需 Anthropic 官方账号也能开始 AI 对话。

详细配置步骤见模型供应商页面。
