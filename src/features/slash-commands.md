# 斜杠命令与文件面板

输入框有三种快捷触发方式，帮助你快速插入命令、引用文件。本页逐一介绍它们的用法与细节。

> 相关基础：[界面导览](/guide/interface-overview)、[快速上手](/guide/quick-start)

---

## 输入框触发提示

当光标停在任务草稿输入框时，占位提示会显示三种触发方式：

> 「向 Claude 提问，输入 @ 添加文件，/ 使用命令，$ 使用技能」

- 输入 `/` —— 打开**命令面板**，浏览并调用斜杠命令或技能
- 输入 `@` —— 打开**文件面板**，将工作区文件附加到消息
- 输入 `$` —— 快速筛选出技能类命令（技能本质上也是斜杠命令的一种，在命令面板中以技能徽标区分）

---

## / 命令面板

### 这是什么

命令面板（Command Palette）是所有**斜杠命令**的统一入口。你可以在这里搜索并选用来自不同来源的命令，包括内置功能命令、你自己定义的用户命令、项目级命令、插件提供的命令，以及各种技能（Skill）。

技能是一类特殊的可调用单元，与普通命令共享同一个面板，通过徽标区分。

### 在哪里打开

在输入框中，光标所在文本段以 `/` 开头、且不含空格或换行时，命令面板自动弹出在输入框上方。

### 怎么用

1. 在输入框中输入 `/`，命令面板立即弹出，显示全部可用命令。
2. 继续输入关键词过滤（例如 `/compact`），列表实时缩小。
   - 对于 `plugin:cmd-name` 格式的插件命令，输入冒号后的部分也能匹配（例如 `/bra` 可以匹配 `/superpowers:brainstorming`）。
3. 用方向键 ↑↓ 上下导航，或直接用鼠标悬停高亮。
4. 按 **Enter** 或 **Tab** 确认选中，也可点击某条命令。
5. 选中后，`/query` 文本自动替换为该命令的标签片段，插入到光标位置。
6. 按 **Esc** 关闭面板（再次输入 `/` 可重新打开）。

无匹配时面板提示「没有匹配的命令」。

**示例**：输入 `/compact` → 选中 compact 命令 → 按 Enter，输入框出现该命令标签，发送后 AI 将压缩当前对话历史。

### 命令来源徽标

每条命令右侧显示两枚彩色徽标，分别标注该命令的**来源**和**类型**。

**来源徽标**（第一枚）：

| 徽标显示 | 颜色 | 含义 |
|---------|------|------|
| user | 蓝色 | 用户个人自定义命令 |
| project | 绿色 | 当前项目级命令（存放于项目内） |
| plugin | 紫色 | 插件提供的命令 |
| 内置 | 灰色 | 应用内置命令 |

**类型徽标**（第二枚）：

| 徽标显示 | 颜色 | 含义 |
|---------|------|------|
| command | 无背景（浅灰文字） | 普通命令 |
| skill | 琥珀色 | 技能（Skill），可调用专项能力 |
| workflow | 玫红色 | 工作流 |

::: tip 0.5.2 起支持双徽标
v0.5.2 起，命令面板每条目同时展示来源与类型两枚徽标，方便你在命令较多时快速定位来源。技能命令的来源徽标会根据是否属于插件自动推导：插件技能显示 **plugin**（紫色），用户技能显示 **user**（蓝色）。
:::

## @ 文件面板

### 这是什么

文件面板（File Palette）让你快速在消息里引用工作区文件，AI 会将被引用文件的内容作为上下文处理。

### 在哪里打开

在输入框任意位置输入 `@`，且 `@` 前面是行首或空格（非词中间），文件面板自动弹出。光标与 `@` 之间不能有空格，否则面板不会触发。

### 怎么用

1. 在输入框中输入 `@`，文件面板弹出，显示工作区所有可索引文件。
2. 继续输入文件名关键词（例如 `@README`），列表按相关度排序过滤：完全匹配 > 文件名前缀 > 路径前缀 > 文件名包含 > 路径包含。
3. 用方向键 ↑↓ 或鼠标选中目标文件，按 **Enter** 或点击确认。
4. 选中后 `@query` 被替换为文件标签片段（显示文件名）；图片文件会直接转为图片片段嵌入消息。
5. 最多显示 50 条匹配结果，超出时提示「还有 {n} 条…」。

**示例**：输入 `@src/main` → 面板过滤出以 `src/main` 开头的文件 → 选中 `src/main.ts` → Enter，消息里出现该文件标签。

::: tip 找不到文件？
文件面板仅显示工作区**已索引**文件。若文件未出现，请确认已打开对应工作区（左侧栏「查看文件」），或文件是否在 .gitignore 等忽略范围内。
:::

## $ 技能触发

技能（Skill）是通过命令面板调用的专项能力单元，在面板中以琥珀色 **skill** 徽标标注。

使用方式：

- 直接输入 `/` 打开命令面板，再输入技能名称关键词筛选；
- 或输入 `$` 后跟技能名作为快捷搜索前缀（`$` 为触发提示，最终仍通过命令面板完成选择）。

选中技能命令后，AI 会调用对应技能执行专项任务。
