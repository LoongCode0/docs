# 模型供应商

**入口：** 侧边栏 → **设置** → **模型供应商**

模型供应商功能让你在不使用 Claude/Codex 官方账号的情况下，通过第三方 API 或自建代理接入模型服务。配置供应商后，可在任务选择器栏中选择对应的模型档位，调用该供应商提供的模型。

---

## 供应商类型

应用将供应商分为三类：

| 类型 | 说明 |
|------|------|
| **原生（OAuth）** | Claude OAuth 与 Codex OAuth，凭据由 CLI 自身登录管理，应用仅管理模型清单与上下文窗口配置 |
| **内置** | 预置的第三方平台，开箱即用，填入 API Key 即可 |
| **自定义** | 手动填写接口地址与 API Key，适用于任意 OpenAI/Anthropic 兼容代理 |

---

## 内置供应商

应用预置了以下 8 家第三方平台，无需手动填写接口地址，直接输入 API Key 即可启用：

| 供应商名称 | 说明 |
|-----------|------|
| deepseek | DeepSeek 平台，支持 Anthropic 与 OpenAI 双接口 |
| z.ai | 智谱 z.ai 平台，仅 Anthropic 接口 |
| bigmodel | 智谱 BigModel 平台，支持 Anthropic 与 OpenAI 双接口 |
| openrouter | OpenRouter 聚合平台，支持 Anthropic 与 OpenAI 双接口 |
| moonshot | 月之暗面 Kimi 平台，支持 Anthropic 与 OpenAI 双接口 |
| minimax | MiniMax 平台，支持 Anthropic 与 OpenAI 双接口 |
| mimo | 小米 Mimo 平台，仅 Anthropic 接口 |
| Qwen | 阿里云通义千问平台，支持 Anthropic 与 OpenAI 双接口 |

---

## 供应商可用性

每个供应商面板右侧会显示两个可用性指示标记：

- **Claude CLI 可用** — 当 Anthropic 接口地址非空时显示
- **Codex CLI 可用** — 当 OpenAI 接口地址非空时显示

仅支持单接口的供应商（如 z.ai、mimo）只对相应 CLI 可用。

## 配置供应商

![模型供应商页面，左侧供应商列表，右侧显示选中供应商的 API Key、接口地址、模型映射等配置字段](...)

### 1. 选择供应商

在左侧供应商列表中点击要配置的供应商名称，右侧显示详情面板。

### 2. 填写 API Key

在 **API Key** 输入框中粘贴从对应平台获取的密钥。

::: warning 请妥善保管 API Key
API Key 是访问模型服务的唯一凭据，泄露后可能导致费用损失或数据风险。不要将 API Key 截图分享、写入版本库，或通过不安全渠道传输。
:::

### 3. 配置接口地址（可选）

内置供应商已预填接口地址，通常无需修改。若平台更换域名或你使用私有部署版本，可在以下字段中修改：

| 字段 | 说明 |
|------|------|
| **Anthropic 接口地址** | 用于 Claude CLI；支持 Anthropic 消息格式的接口根地址 |
| **OpenAI 接口地址** | 用于 Codex CLI；支持 OpenAI Chat Completion 格式的接口根地址 |

::: tip
两个接口地址不能同时为空，否则保存时会提示错误。
:::

### 4. 配置模型映射

模型映射将应用的三个档位槽（**Opus**、**Sonnet**、**Haiku**）与供应商实际提供的模型名称对应起来。

| 档位 | 对应角色 |
|------|---------|
| Opus | 复杂任务 |
| Sonnet | 常规任务 |
| Haiku | 轻量任务 |

在每个档位的下拉列表中选择想要绑定的具体模型。若某档位留空（显示"未设置"），则该档位在此供应商下不可用。

### 5. 配置上下文窗口（可选）

**上下文窗口**字段允许手动指定模型支持的 Token 数量上限，用于上下文使用率显示与自动压缩触发判断。留空则使用应用内置的默认值（v0.5.2 起出厂预设了主流模型的默认值；未匹配到的模型回退到 200,000 Token）。

## 添加自定义供应商

若你的目标平台不在内置列表中（例如自建 OpenAI 兼容代理），可添加自定义供应商：

1. 点击左侧供应商列表底部的"添加"按钮，列表中出现名为**新供应商**的条目。
2. 点击名称旁的编辑（铅笔）图标，输入供应商的显示名称。
3. 按 [配置供应商](#配置供应商) 步骤填写接口地址、API Key 与模型映射。

自定义供应商可通过列表底部的删除按钮移除（需二次确认）；内置供应商与 OAuth 供应商不可删除。

---

## Codex 内置中转

若已安装 Codex CLI，部分支持 OpenAI 接口的供应商会出现**经内置中转使用（Codex）** 开关。开启后，Codex CLI 的请求将经由应用内置的协议转换中转（Chat Completion → Response Stream）后转发至该供应商，从而让 Codex CLI 使用非官方接口。
